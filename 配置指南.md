# 小红书 AI Agent 配置指南

## 环境配置

### 1. 安装依赖

```bash
cd /Users/erich/Desktop/xiaohongshu-agent

# 安装 Python 依赖
pip install -r requirements.txt
```

### 2. 配置 API Key

#### 方式一：环境变量

在 `~/.zshrc` 或 `~/.bashrc` 中添加：

```bash
# 选择一个 LLM 提供商
export LLM_PROVIDER=openai

# OpenAI
export OPENAI_API_KEY=your_key_here

# 或 Anthropic (Claude)
# export ANTHROPIC_API_KEY=your_key_here
# export LLM_PROVIDER=anthropic

# 或智谱 GLM
# export ZHIPU_API_KEY=your_key_here
# export LLM_PROVIDER=zhipu

# 或 Kimi
# export KIMI_API_KEY=your_key_here
# export LLM_PROVIDER=kimi

# 或 Google Gemini
# export GEMINI_API_KEY=your_key_here
# export LLM_PROVIDER=gemini

# 或 Minimax
# export MINIMAX_API_KEY=your_key_here
# export LLM_PROVIDER=minimax
```

然后执行：
```bash
source ~/.zshrc
```

#### 方式二：交互式配置

```bash
cd /Users/erich/Desktop/xiaohongshu-agent

# 启动配置向导
./start.sh setup
# 或
python3 src/setup.py
```

按提示选择 LLM 提供商并输入 API Key。

#### 方式三：界面配置

```bash
./start.sh gui

# 选择 S → 1 选择提供商
# 选择 S → 2 配置 API Key
```

### 3. 配置 MCP

确保小红书 MCP 服务正在运行：

```bash
# 检查 MCP 服务
curl http://localhost:18060/health

# 如果未运行，启动它
cd /Users/erich/Desktop/xiaohongshu-mcp-darwin-arm64
./xiaohongshu-mcp-darwin-arm64 &
```

## 使用方法

### 快速启动

```bash
cd /Users/erich/Desktop/xiaohongshu-agent

# 交互式界面
./start.sh gui

# 搜索
./start.sh search -k "AI Agent"

# 统计
./start.sh stats

# AI 对话
./start.sh chat

# 查看配置
./start.sh config
```

### 交互式界面菜单

```
═══════════════════════════════════════
          主菜单
═══════════════════════════════════════
  [1]  🔍 搜索热门帖子
  [2]  📝 创建并发布帖子

  [6]  🤖 AI 对话
  [S]  ⚙️ 系统设置
```

### 系统设置 (S)

- **1**: 选择 LLM 提供商
- **2**: 配置 API Key
- **3**: 配置 MCP 地址
- **4**: 配置权限
- **5**: 查看当前配置
